Cómo Jugar

  • Para cada nuevo número, recuerda el número que apareció {{ n_back }} elementos atrás.
  • Si el número actual es el mismo que el número {{ n_back }} elementos antes, presiona la barra espaciadora.
  • Si no es el mismo, no presiones nada.
